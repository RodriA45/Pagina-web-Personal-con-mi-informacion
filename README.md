# Mi Sitio Personal - Rodrigo Antunez

Este es mi primer sitio web personal, desarrollado como parte del **Trabajo Práctico N°1** para la materia **Programación III**. El objetivo del proyecto es aplicar conocimientos fundamentales de desarrollo web para crear una carta de presentación digital, organizada y visualmente atractiva.

## 🚀 Vista Previa
Puedes ver el sitio en vivo a través de GitHub Pages:
👉 [Ver Sitio Web](https://rodria45.github.io/Pagina-web-Personal-con-mi-informacion/)

## 🛠️ Tecnologías Utilizadas
- **HTML5**: Estructura semántica del contenido.
- **CSS3**: Diseño responsivo y estilos personalizados utilizando variables y Flexbox.
- **Google Fonts**: Tipografía moderna (Roboto).

## 📂 Estructura del Proyecto
El sitio está organizado en diferentes secciones accesibles desde el menú de navegación:
- **Inicio**: Presentación general y frase motivadora.
- **Sobre mí**: Reflexión personal y perfil académico.
- **Proyectos**: Galería de trabajos actuales y futuros, incluyendo:
  - Sitio Web Personal (Completado).
  - App de Gestión de Alumnos (En desarrollo).
  - Gestor de Clientes para Taller de Motos (Terminado).
- **Gustos y Hobbies**: Secciones dedicadas a mis intereses personales para brindar un perfil más humano.

## 💻 Instalación y Uso
Si deseas ejecutar este proyecto localmente:
1. Clona el repositorio: `git clone https://github.com/rodria45/Pagina-web-Personal-con-mi-informacion.git`
2. Navega al directorio del proyecto.
3. Abre el archivo `index.html` en tu navegador preferido.
